Web Scraping

Es un proceso dentro de la ciencia de datos, que permite extraer informacion de sitios Web, normalmente usados en el lenguaje de programacion python, simulando la navegación en la web, ya sea usando protocolo manual de http o incrustando un navegador en una aplicacion.


Para el desarrollar parte del proceso de la tarea se procede hacer peticiones con la biblioteca Requests de python y se simula la navegacion con BeautifulSoup (bs4), para este caso en particular como la páginas a realizar scrapign son estaticas, es decir, no estan desarrolladas bajo un framework de JavaScript, se hace sencillo realizar el proceso de scraping.
